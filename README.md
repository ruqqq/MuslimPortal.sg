# BarnamijSG
Events for Muslim community in Singapore

COLLECTION FIELDS
<p>eventType (dropdown: class, talk, social)</p>
<p>title</p>
<p>description</p>
<p>speaker (optional according to eventType)</p>
<p>when:{</p>
	<p>isRecurring</p>
	<p>dateStart</p>
	<p>dateEnd (optional, if isRecurring)</p>
	<p>timeStart</p>
	<p>timeEnd</p>
	<p>timeStamp</p>
<p>}</p>
<p>where: {</p>
<p>	address</p>
<p>	venue</p>
<p>	area</p>
<p>}</p>
<p>fee (optional, default 0l)</p>
<p>tags</p>

FILTERABLE BY
eventType (dropdown)
area (dropdown)
